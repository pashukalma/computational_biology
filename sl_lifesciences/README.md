This site focus on the application of statistical modeling and machine learning to medical research. We primarily utilize datasets from the "Elements of Statistical Learning" (ESL) to demonstrate predictive analytics and sequence analysis.

1. **Clinical Prediction & Risk Analysis** on modeling clinical outcomes based on patient measurements:
2. **Cancer Genomics & Microarrays** Explore high-dimensional data where the number of features (genes) far exceeds the number of samples
3. **Bioinformatics Algorithms** Gene Search & Alignment; Search Heuristics: Mentions tools like BLAST (Basic Local Alignment Search Tool).
4. **Exploratory Data Analysis (EDA)** ESL Mixture Dataset analysis of a classic "mixture" dataset used to demonstrate supervised learning. It compares different classification boundaries and utilizes Multidimensional Scaling (MDS) and Random Forest Classifiers to visualize and predict class labels in a 2D predictor space.
